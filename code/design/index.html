设计编译

通过算法把设计稿 PSD、Sketch 翻译为计算机语言
适用于Ios、Android、H5 等平台

立即体验

应用场景

  • 设计还原
    把设计稿转为UI代码，适用绝对布局及autolayout。
  • 多终端适配
    能提供Ios、Android、H5等平台语言的支持，同时能兼容各分辨率屏幕。
  • 视觉校对
    与规范文档智能比对，筛选出不符合要求的设计，并对问题进行重要程度排序
  • 快速验证
    上传设计稿即生成在线地址，以快速验证在各手机端的表现
  • 色彩搭配
    提取出页面中所以用到的色彩占比，方便设计查看色彩的搭配比例，配色方案智能推荐与测试
  • 辅助创作
    帮助设计师排查错误，减少重复工作

方案优势

  • 多平台适应

    设计数据抽象DSL，能输出Ios、Android、H5所需语言

  • 多格式支持

    支持sketch、psd两种格式代码转换，满足不同的设计需求

  • 数据收集

    自动提取设计稿中的组件，用于规范校验及组件库生成

  • 拟人化结构

    通过AI训练，自动识别页面组件，从而输出更合理页面结构

  • 精确还原

    程序解析设计参数一对一翻译，确保设计稿100%精确还原

  • 图形处理

    识别设计图层中的图形元素，把同一单元的图形，智能合并为图片
