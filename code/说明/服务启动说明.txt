result:用于静态网页启动服务
启动方法：跳转到该路径下。启动命令：node app

upload:用于sketch文件的上传，生成对应的uuid的url服务
启动方法：跳转到该路径下。启动命令：npm run start


pm2守护进程安装方法:
pm2 start ./bin/www
pm2 start ./bin/www --watch 含监听


//安装babel后
还有npm start或者npm run start



安装unzip：cnpm i unzip --save



package.json中：(二者只能写一个，注释也不行，否则报错)
"start": "babel-node ./bin/www --presets es2015,stage-2"   支持es6语法    npm run start
"start": "node ./bin/www"            普通es5写法   npm run start
