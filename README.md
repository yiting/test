# 视觉编译
相信大家都知道，在我们的研发流程中，有一个环节“UI还原”，就是把设计语言转为编程语言。自从灵长动物诞生以来，猿人们大多是一边看设计师的各种标注或参数然后翻译为不同环境的代码，以实现UI还原。基于人工操作误差总会存在的，100%的UI还原需要消耗大量时间，而且这些重复这些体力活实在没有成就感，我们更希望用更多的时间写逻辑或做一些有创造性的工作。
